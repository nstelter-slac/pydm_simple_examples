# pydm_simple_examples
simple examples of python code using pydm (originally for a TIDTalk)

```
# run ./pydm-testing-ioc from pydm examples dir in another terminal

# example of pure python pydm screen
pydm simple_display_python_only.py

# example of pure ui python screen (made in QtDesigner)
pydm simple_display.ui

# example of python screen reading in and adding processing to ui file (common way to make screens)
pydm simple_display_python_reads_in_ui.py
```
